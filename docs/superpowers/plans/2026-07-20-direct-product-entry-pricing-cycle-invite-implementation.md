# 官网立即使用、报价切换与邀请码入口实施计划

## 实施步骤

1. 将导航、首屏、报价卡、开始使用区和最终 CTA 的操作统一为产品地址链接。
2. 删除公开页面的试用表单状态、字段校验、提交请求和弹窗渲染，保留代理合作表单。
3. 为报价区增加购买周期状态和年付/永久切换，卡片布局由四列调整为当前周期两列。
4. 更新中国站报价月度提示，保留既有年度价和永久买断价。
5. 删除中国站首页的邀请码服务端跳转，只在代理合作页客户端读取并预填参数。
6. 同步官网总规格和旧邀请码设计的替代关系。
7. 增加静态回归脚本，验证直达链接、表单移除、报价切换和邀请码入口边界。
8. 运行回归脚本、CMS 类型检查、完整构建和浏览器桌面/移动端验收。
9. 仅提交本次相关文件，不纳入新闻文案、Logo 和临时目录变更。

npm run site:verify-direct-entry
