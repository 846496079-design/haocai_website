# 浅色科技视觉与四方案定价设计

## 目标

将官网从深色混搭调整为统一的雾白科技视觉；替换最新横版 SVG Logo；以四张同级方案卡展示年付与永久买断；重做产品页导航，使其与页面协调。

## 视觉系统

- 页面基底使用雾白、冷灰与浅青，正文保持深蓝灰可读性。
- 背景使用低对比浅青网格，并叠加低频漂移光点；动画只作用于装饰层，遵循减少动态偏好。
- 卡片统一为半透明白色、细浅青边框、轻阴影和 20px/28px 圆角。
- CTA 使用青蓝；永久买断使用金色边框和标签，仅作为商业强调，不改变全站主色。
- 导航、页脚和公司品牌区使用 `zds-logo-h-*-vector.svg`。

## 产品页导航

- 移除左侧固定“本页导航”。
- 在产品首屏下放置一条横向锚点导航，桌面端居中展示，移动端可横向滑动。
- 当前锚点保持青蓝底色，其余项采用白色玻璃按钮；导航不遮挡首屏内容。

## 首页和定价

- 首页保持转化结构：主视觉、开始使用、定价、案例规模与反馈、新闻、站内导航。
- 定价将两个套餐的年付和永久买断拆为四张同级方案卡：小规模年付、小规模永久买断、一般纳税人年付、一般纳税人永久买断。
- 永久买断卡片以金色标签、长期使用文案和更强边框强调；年付卡保留价格与每日成本。

## 验收

1. CN、JP、HK 页面均使用浅色科技变量与同一套卡片、按钮和背景规则。
2. 最新 SVG Logo 在导航、页脚和公司页正常显示。
3. 产品页无左侧固定导航，存在可用横向锚点导航。
4. 定价区域可见四张同级方案卡，其中两张永久买断卡显著但不遮挡年付信息。
5. 390px、768px、1440px 无横向溢出，`npm run build` 成功。
