# 浅色科技视觉与四方案定价设计

## 目标

将官网调整为统一的浅色科技视觉；替换最新横版品牌 Logo；以四张同级方案卡展示年付与永久买断；移除产品页内导航，并消除案例页重复标题。

## 视觉系统

- 页面基底使用 `#F7F9FC`、冷灰与蓝紫表面色，正文使用 `#18243D`，二级文字使用 `#667085`。
- 背景使用低对比蓝紫网格，并叠加低频漂移光点；动画只作用于装饰层，遵循减少动态偏好。
- 卡片统一为白色或 `#EEF2FF` 表面、细蓝色边框、轻阴影和 20px/28px 圆角。
- CTA 使用 `#4F7BFF → #6C5CFF` 渐变；永久买断使用金色边框和标签，仅作为商业强调，不改变全站主色。
- 导航、页脚和公司品牌区使用 `official-logo-*.png`。

## 产品页导航

- 移除产品页所有页内锚点导航，避免与顶部主导航重复。
- 首屏仅保留产品价值说明、主要 CTA 与产品截图。

## 案例页内容层级

- 案例页首屏负责表达“客户反馈”主题。
- 删除首屏后重复的“客户反馈 / 来自真实工作场景的使用反馈”二级标题与说明。
- 服务规模与匿名案例卡片直接承接首屏，避免同义标题连续出现。

## 首页和定价

- 首页保持转化结构：主视觉、开始使用、定价、案例规模与反馈、新闻、站内导航。
- 定价将两个套餐的年付和永久买断拆为四张同级方案卡：小规模年付、小规模永久买断、一般纳税人年付、一般纳税人永久买断。
- 永久买断卡片以金色标签、长期使用文案和更强边框强调；年付卡保留价格与每日成本。

## 验收

1. CN、JP、HK 页面均使用浅色科技变量与同一套卡片、按钮和背景规则。
2. 最新品牌 Logo 在导航、页脚和公司页正常显示。
3. 产品页不显示页内导航。
4. 定价区域可见四张同级方案卡，其中两张永久买断卡显著但不遮挡年付信息。
5. 案例页仅保留一组客户反馈标题，服务规模区紧接首屏内容。
6. 390px、768px、1440px 无横向溢出，`npm run build` 成功。
