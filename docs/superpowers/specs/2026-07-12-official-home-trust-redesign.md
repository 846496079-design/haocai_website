# 账大师官网首页可信 AI 财务系统重设计

## 背景

首页已经具备真实产品截图、清晰定价和多端入口，但视觉中同时叠加网格、蓝紫渐变、玻璃材质、光晕、悬浮卡片和多组动效，降低了企业财税产品应有的可信度与信息效率。

本次仅重设计官网首页的展示与转化层级，不改动线索提交、价格数据、路由、新闻内容或既有多语言文案事实。

## 设计目标

- 建立“可信、清楚、低门槛”的 AI 财务产品首屏印象。
- 让访客按“价值主张 → 产品如何使用 → 价格 → 真实反馈 → 延伸内容”理解页面。
- 保留真实产品截图、公众号、网页版与 Mac 版入口，以及现有定价事实。
- 保持 CN、JP、HK 三个站点共享首页结构，文案继续从既有内容模型读取。
- 让移动端所有主要可操作入口达到至少 44px 的触控高度，无横向溢出。

## 视觉方向

设计语言采用克制的可信 AI 财务系统风格：冷白背景、深蓝黑文字、单一账大师品牌蓝作为强调色。

- 移除首页的渐变文字、玻璃拟态、背景大面积光晕、装饰性网格漂移和非必要悬浮卡片。
- 使用细分隔线、克制的浅蓝表面和有意义的产品界面截图建立层次。
- 保留一组轻量进入动效，但默认内容必须可见；在减少动态偏好下完全静止。
- 统一使用现有圆角尺度，不引入新的字体或第三方依赖。

## 页面结构

1. 首屏：一句价值主张、一段说明、两个清晰行动入口、三项可验证的产品承诺和真实产品截图。
2. 使用方式：公众号、网页版和 Mac 版三个入口，压缩为快速选择区。
3. 定价：保留现有套餐、功能与说明，强化主要年付方案，降低无关视觉噪声。
4. 客户反馈：保留匿名案例和行业分布，不把未经核验的规模数据当作效果承诺。
5. 新闻与延伸：保留现有新闻和站内导航卡片，统一排版与交互反馈。

## 实现边界

- 只修改首页共享组件、首页所需的全局样式和本规格文档。
- 不修改 API 地址、提交字段、套餐金额、产品功能事实、新闻内容或其他页面的信息架构。
- 不纳入工作区已有的未提交图片、公司资料或新闻资源变动。

## 可访问性与性能

- 顶部导航、主要 CTA、移动端菜单和首页文字链接的可点击区域不小于 44px。
- 下拉站点菜单补充 `aria-expanded` 与 `aria-controls`。
- 首屏和各区块不能依赖 JavaScript 动画才显示内容。
- 所有首页动画仅使用 `opacity` 与 `transform`，并提供 `prefers-reduced-motion` 兜底。
- 保留 Next Image 优化与现有真实产品图片，不增加网络图片或依赖。

## 验收

- `/cn/`、`/jp/`、`/hk/` 首页能正常渲染，且共享结构保持一致。
- 在 390px、768px 与 1440px 宽度下无横向滚动，主要入口可触达。
- `npm run build` 通过；若 `npm run lint` 仍不可运行，明确记录现有 `eslint` 缺失问题，不为本次视觉重设计额外安装依赖。
- 人工检查无中文乱码、无公开内部开发措辞、无 emoji。
