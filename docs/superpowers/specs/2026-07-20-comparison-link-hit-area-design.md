# 完整功能对比表入口点击区域修复设计

## 问题与根因

首页报价区下方的“查看完整功能对比表”文字位于一个高度为 44px 的链接中，但链接外层使用 `-mt-8` 向上移动 32px，与前一个 `position: relative` 的报价 section 重叠。浏览器命中测试显示，文字顶部、中心和底部均被报价 section 截获，只有链接最底部的小部分区域可以点击。

## 采用方案

- 保留现有视觉位置、文字、箭头、链接地址和 44px 点击高度。
- 仅在链接外层增加 `relative z-10`，让整个链接位于报价 section 的命中层级之上。
- 不删除 `-mt-8`，避免入口整体下移并改变当前页面节奏。
- 不移动链接到报价组件内部，避免扩大首页与产品中心共用组件的变更范围。

## 方案比较

1. 提升链接容器层级（采用）：变更最小，保持现有视觉位置，并恢复完整命中区域。
2. 删除负间距：能够消除重叠，但入口会整体下移 32px，改变当前视觉节奏。
3. 将链接移入报价 section：结构上消除跨区重叠，但会扩大组件边界和回归范围。

## 验收标准

1. 链接仍显示在当前视觉位置，文字和箭头不发生明显位移。
2. 链接保持至少 44px 高的触控区域。
3. 文字顶部、中心、底部以及链接顶部、底部的命中元素均为 `/cn/product/#comparison` 链接。
4. 点击文字可以进入产品中心的 `#comparison` 功能对比区域。
5. 首页报价卡、产品中心报价区及其他页面不发生结构或样式变化。
